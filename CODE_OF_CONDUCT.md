# Community Code of Conduct

Aegis GMS welcomes constructive contributions, bug reports, and technical discussion.

Participants must:

- Treat others with respect.
- Critique code and ideas without attacking people.
- Avoid harassment, threats, discrimination, or deliberate disruption.
- Protect private information and credentials.
- Report security vulnerabilities privately.

Maintainers may edit, hide, lock, or remove contributions that violate these expectations and may restrict participation when necessary to protect the project and its community.
